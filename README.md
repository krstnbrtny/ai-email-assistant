# AI Email Assistant

## Overview

An AI-powered email automation workflow built using n8n, OpenAI GPT-4.1 Mini, Gmail, and Telegram.

This workflow automatically monitors incoming emails, generates AI-powered summaries, determines priority levels, recommends actions, and sends notifications to Telegram.

## Workflow

Gmail Trigger → OpenAI GPT-4.1 Mini → Telegram

## Technologies Used

* n8n
* Gmail API
* OpenAI GPT-4.1 Mini
* Telegram Bot API

## Features

* Automatic email monitoring
* AI-generated summaries
* Priority classification
* Action recommendations
* Telegram notifications

## Example Output

Summary: Customers are unable to complete their orders due to a website checkout failure.

Priority: HIGH

Action: Investigate and resolve the checkout issue immediately.
